# Frontend Workspace

The main project README lives at the repository root:

- see [`../README.md`](../README.md) for the full hackathon overview
- run all app commands from this `frontend/` directory

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Required Environment Variables

```bash
ANTHROPIC_API_KEY=your_anthropic_key
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_long_random_session_secret
```

Open `http://localhost:3000` after starting the dev server.
